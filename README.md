# 🎨 Колеровщик

Веб-приложение для подбора колера краски «на глаз». Загрузите фото эталонного
цвета и фото вашего текущего замеса, тапните пипеткой по любой точке каждого
снимка — приложение сравнит цвета и подскажет, какой колер добавить, чтобы
попасть в нужный оттенок.

Работает прямо в браузере, без установки, полностью офлайн (после первой загрузки
шрифтов). Один HTML-файл, без сборки и без бэкенда.

## ✨ Возможности

- **Пипетка по фото** — цвет считывается напрямую из пикселей снимка.
- **Анализ разницы** — приложение раскладывает отклонение по каналам R, G, B и
  отдельно по яркости.
- **Текстовые подсказки** — «добавьте каплю жёлтого», «разбавьте белой базой»,
  «приглушите оранжевым» и т. д.
- **Светлая и тёмная тема** — переключатель в шапке, выбор запоминается.
- **Установка на домашний экран** (iPhone/Android) — запускается в полный экран
  как обычное приложение, со своей иконкой.

## 🚀 Как пользоваться

1. Откройте приложение в браузере.
2. Нажмите **«Загрузить эталон»** и выберите фото нужного цвета.
3. Нажмите **«Загрузить краску»** и выберите фото вашего замеса.
4. Тапните по любой точке на каждом фото — под ними появятся образцы и RGB-коды.
5. Прочитайте рекомендацию в блоке внизу.

> ⚠️ Делайте тест на небольшом объёме (~100 мл) и обязательно высушите мазок
> феном перед оценкой — мокрая краска всегда темнее сухой.

## 📱 Установка на iPhone

1. Откройте ссылку на приложение в **Safari**.
2. Нажмите кнопку **«Поделиться»**.
3. Выберите **«На экран Домой»**.

Приложение появится иконкой на рабочем столе и будет открываться в полноэкранном
режиме. Доступ к камере работает только по `https://` (то есть с GitHub Pages,
а не при открытии файла локально).

## 🛠️ Технологии

Чистый HTML + CSS + JavaScript (Vanilla JS), без зависимостей и сборки. Цвет
читается через `Canvas 2D` и `getImageData`. Тема хранится в `localStorage`.

## 📋 Запуск локально

Просто откройте `index.html` в браузере. Учтите: при локальном открытии
(`file://`) на iPhone камера будет недоступна — доступна только галерея.
Для полноценной работы разместите файл на хостинге с `https://`
(например, GitHub Pages).

## ⚖️ Как читать рекомендации

Приложение сравнивает цвета в модели **RGB** — так, как их видит камера. Реальное
смешивание красок субтрактивное и нелинейное, поэтому подсказки задают
**направление** корректировки, а не точную рецептуру. Для надёжного результата:

- фотографируйте эталон и краску **при одинаковом освещении** (лучше дневном
  рассеянном);
- добавляйте колер **по капле** и пересматривайте после каждой просушки.
